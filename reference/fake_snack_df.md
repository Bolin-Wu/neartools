# Fake SNAC-K data

A randomized SNAC-K data set, with metadata preserved.

## Usage

``` r
fake_snack_df
```

## Format

A data frame with 15 variables and 3365 random obs.

## Details

Random values are generated by
[`rnorm()`](https://rdrr.io/r/stats/Normal.html) and
[`sample()`](https://rdrr.io/r/base/sample.html).
